WHAT IS HTML?

HTML (HiperText Markup Language) is the basic component of the web which allows you to insert content and establish the basic structure of a website. Therefore, it serves to give meaning and organize the information of a web page. Without this, the browser would not be able to display text as elements or upload images and other content. Hypertexts are sets of connected elements. These can be words, images, videos, document, etc. When connected, they form a network of information that allows data communication, organizing knowledge and saving information. When visiting a simple web page, you may notice that there are different distributions and sizes for titles, paragraphs, images, videos, and any other element. This structure is established through HTML. At the beginning of the web, it was common to find sites only containing simple texts and images, with basic structure and without stylizations. However, these days, you will hardly find sites that have only HTML elements. Therefore, we can consider HTML the "skeleton" of your page. Imagine then that in addition to the skeleton, it is necessary to have the body. For this, we then have the CSS and JavaScript languages, which together with HTML, form the basis for all current websites.
